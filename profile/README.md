## Retailpulses GK

Japan-based ecommerce operations and software. Cross-platform listing, inventory, inquiry management for Mercari, Rakuten, Amazon Japan.

### Active Repos

| Area | Repo | Purpose |
|------|------|---------|
| Mercari | [mercariops](https://github.com/retailpulses/mercariops) | Listing, timesale, tickets, CSV, copywriting |
| Rakuten | [rakutenops](https://github.com/retailpulses/rakutenops) | Listing, activation, pricing, CSV, images |
| Amazon | [amazonops](https://github.com/retailpulses/amazonops) | Listing sync, price mgmt, inventory flatfile |
| Workers | [workers](https://github.com/retailpulses/workers) | Cloudflare Workers monorepo |
| Catalog | [CatalogSync](https://github.com/retailpulses/CatalogSync) | Cross-platform catalog sync |
| Orders | [OrderMgmt](https://github.com/retailpulses/OrderMgmt) | Order management |
| Inquiries | [inquiry-automation](https://github.com/retailpulses/inquiry-automation) | Customer inquiry automation |
| Tickets | [ticket-handling](https://github.com/retailpulses/ticket-handling) | Mercari ticket management |
| Listings | [boutique-listing](https://github.com/retailpulses/boutique-listing) | Boutique listing web tool |
